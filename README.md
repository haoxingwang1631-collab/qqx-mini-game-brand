## 📋 项目简介

本项目是一套完整的「QQ媒体 × 微信小游戏」商业化运营作战系统，面向2-3人小团队（运营+销售），覆盖从行业洞察、客户筛选、竞品分析、方案输出、商机跟进到投后复盘的全链路闭环工作流。

## 🚀 快速上手

### 第1步：通读SOP
打开 `01_工作流SOP主文档.md`，了解8大模块的完整定义、执行步骤和协作机制。

### 第2步：配置工具
将 `02_配套Excel模板/` 中的表格导入到腾讯文档/企微文档（后续支持MCP自动同步），设置好权限。

### 第3步：启动执行
按照SOP中定义的执行节奏开始运转：
- **每周一**：推送企微群资讯快报
- **每周二**：产出情报周报 + 更新客户池
- **每周四**：商机进展同步
- **每双周**：竞品动态更新
- **每月末**：月度复盘 + 客户分层更新

### 第4步：持续迭代
每月底根据实际使用体验，优化模板和流程。

## ⚠️ 注意事项

1. **数据安全**：商机追踪表含客户敏感信息，请控制访问权限
2. **版本管理**：所有模板文件修改后请更新版本号
3. **信息时效**：竞媒调研报告建议每季度更新一次，确保数据时效性
4. **协作纪律**：按SOP规定的节奏执行，避免信息断层
